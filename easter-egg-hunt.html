A Short Brief

Battleship Web Game themed as Easter Egg Hunt. The goal of the game is to find all the missing easter eggs. This game can be played with a friend.

Project Details

The main challenge of the development process relied on taking responsibility over the art and audio department at the same time I developed all game functionalities.
The main challenges and achievements of the project were:

  • - Implement the algorithm to place the hidden easter eggs. You can find the code here
  • - Use of modern JavaScript syntax to develop the game with an object oriented solution
  • - Make sure the gameplay flow was correclty implemented using jQuery and CSS
  • - Use the browser local storage to store the highscores obtained by the player
  • - Generate art and audio assets for the game
